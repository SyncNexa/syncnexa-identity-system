# Getting Started — Overview

Welcome to **SyncNexa Identity (SAuth 1.0)**! This is a lightweight SAuth 1.0 identity system designed for seamless third-party app integration.

## What is SyncNexa Identity (SAuth 1.0)?

SyncNexa Identity (SAuth 1.0) provides:

- **SAuth Authorization Code Flow** for secure app authorization
- **JWT-based access tokens** for API access
- **User consent screens** for transparent permission management
- **Flexible scopes** for granular data access

## Key Features

- ✅ SAuth protocol implementation
- ✅ Consent-based user approval
- ✅ Multiple scope permissions
- ✅ RESTful API endpoints
- ✅ Session & JWT support

## Before You Start

- Have your **app credentials** (APP_ID, CLIENT_ID, CLIENT_SECRET) ready
- Set up a **redirect URI** for your application
- Understand SAuth basics
